About

Bitcoin Core is an open source project which maintains and releases Bitcoin client software called “Bitcoin Core”. It is a direct descendant of the original Bitcoin software client released by Satoshi Nakamoto after he published the famous Bitcoin whitepaper. Bitcoin Core consists of both “full-node” software for fully validating the blockchain as well as a bitcoin wallet. The project also currently maintains related software such as the cryptography library libsecp256k1 and others located at GitHub. Anyone can contribute to Bitcoin Core.

The power of open source is that it create a community around Bitcoin. There is no power heirarchy which puts everyone on the same lever. People wants to have a secure system that limits fraud and theft of cryptocurrencies, with an opensource platfrom you can pool resources and opinions from around the world.

Blockchain is a new database for transactions. The way all users that "mine" cryptocurrencies can contribute to the transfer of money anonymously around the world. Its is different from traditional Database Management Systems because there is not central hub for storing all the data. Their is no way to curropt the data as it is distributed are the world on thousands of computers.
